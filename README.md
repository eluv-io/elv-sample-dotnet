# elv-sample-dotnet
dotnet runtimes for the Eluvio content fabric

## install .net
https://learn.microsoft.com/en-us/dotnet/core/install/

## building
dotnet build elv-sample-dotnet.sln  -c Debug

## clean
dotnet clean elv-sample-dotnet.sln
